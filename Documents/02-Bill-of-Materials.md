# Bill of Materials (BOM) - Hardware Components

This document lists all the hardware components needed to set up the Park Smart Monitor system. Use this list to estimate costs and purchase the necessary equipment.

## Required Hardware Components

### 1. Security Cameras
**Purpose:** To monitor streets and detect illegally parked vehicles

**Specifications:**
- **Type:** IP Camera (Internet Protocol Camera) or Web Camera
- **Resolution:** Minimum 1080p (Full HD) or higher (2MP, 4MP recommended)
- **Features Needed:**
  - Night vision capability (for 24/7 monitoring)
  - Weatherproof housing (for outdoor installation)
  - Wide viewing angle (to cover more area)
  - Motion detection (optional but helpful)
  - Remote access capability

**Quantity:** 1 or more cameras per monitoring location
- Each camera covers one street or area
- You can add more cameras as needed

**Estimated Cost:** ₱3,000 - ₱15,000 per camera
- Basic IP cameras: ₱3,000 - ₱8,000
- High-quality cameras with night vision: ₱8,000 - ₱15,000

**Installation Requirements:**
- Mounting bracket or pole mount
- Power supply (AC adapter or PoE - Power over Ethernet)
- Network cable (if using wired connection) or Wi-Fi capability

---

### 2. Computer or Server
**Purpose:** To run the monitoring software and store data

**Minimum Requirements:**
- **Processor (CPU):** Intel Core i3 or AMD equivalent (4th generation or newer)
- **Memory (RAM):** 4 GB minimum, 8 GB recommended
- **Storage:** 100 GB free space minimum (for database and images)
- **Operating System:** Windows 10/11, Linux, or macOS
- **Internet Connection:** Required for AI service and SMS sending

**Recommended Specifications:**
- **Processor:** Intel Core i5 or better
- **Memory:** 8 GB RAM or more
- **Storage:** 500 GB or more (SSD recommended for faster performance)
- **Network:** Wired Ethernet connection (more stable than Wi-Fi)

**Options:**
1. **Desktop Computer** - Best for permanent installation
   - Estimated Cost: ₱15,000 - ₱30,000
2. **Laptop Computer** - Portable option
   - Estimated Cost: ₱20,000 - ₱40,000
3. **Mini PC** - Compact and energy-efficient
   - Estimated Cost: ₱10,000 - ₱25,000

**Note:** The computer must be running 24/7 for continuous monitoring.

---

### 3. Internet Connection
**Purpose:** To connect cameras, send SMS messages, and use AI service

**Requirements:**
- **Speed:** Minimum 5 Mbps download, 2 Mbps upload
- **Type:** Broadband internet (DSL, Fiber, or Cable)
- **Stability:** Reliable connection (avoid frequent disconnections)
- **Data Allowance:** At least 50 GB per month (for image uploads and AI processing)

**Estimated Monthly Cost:** ₱1,000 - ₱2,500
- Depends on internet service provider and plan

**Important:** A stable internet connection is critical because:
- Cameras need internet to send images
- AI service requires internet to analyze images
- SMS service needs internet to send text messages

---

### 4. Network Equipment (If Using Wired Cameras)

**Network Switch or Router**
- **Purpose:** To connect multiple cameras to the network
- **Type:** Gigabit Ethernet switch (8-port or 16-port)
- **Quantity:** 1 per location (if using multiple cameras)
- **Estimated Cost:** ₱1,500 - ₱5,000

**Network Cables (Ethernet Cables)**
- **Type:** Cat5e or Cat6 Ethernet cables
- **Length:** Depends on distance from camera to switch/router
- **Estimated Cost:** ₱50 - ₱200 per meter

**Power over Ethernet (PoE) Switch** (Optional but Recommended)
- **Purpose:** Powers cameras through network cable (no separate power adapter needed)
- **Estimated Cost:** ₱3,000 - ₱8,000

---

### 5. Power Supply and Backup

**Uninterruptible Power Supply (UPS)**
- **Purpose:** Keeps system running during power outages
- **Capacity:** 500VA - 1000VA (enough for computer and 1-2 cameras)
- **Estimated Cost:** ₱3,000 - ₱8,000
- **Important:** Prevents data loss and keeps cameras working during brownouts

**Power Adapters for Cameras**
- If cameras don't use PoE, each camera needs a power adapter
- Usually included with camera purchase
- Estimated Cost: ₱200 - ₱500 each (if needed separately)

---

### 6. Mounting Hardware

**Camera Mounts**
- **Type:** Wall mount, pole mount, or ceiling mount
- **Material:** Weatherproof and rust-resistant (stainless steel or aluminum)
- **Estimated Cost:** ₱500 - ₱2,000 per mount

**Installation Hardware**
- Screws, bolts, cable clips, weatherproof boxes
- Estimated Cost: ₱500 - ₱1,500 per camera installation

---

## Optional but Recommended Components

### 7. External Storage (For Image Backup)
- **Purpose:** Store captured images for longer periods
- **Type:** External hard drive (1 TB or larger)
- **Estimated Cost:** ₱3,000 - ₱6,000

### 8. Monitor/Display
- **Purpose:** To view the dashboard and manage the system
- **Type:** Any computer monitor or TV with HDMI input
- **Size:** 19 inches or larger
- **Estimated Cost:** ₱3,000 - ₱10,000

### 9. Keyboard and Mouse
- Standard computer keyboard and mouse
- Estimated Cost: ₱500 - ₱2,000

---

## Software and Services (Not Hardware, but Required)

### 10. SMS Service Subscription
- **Service:** PhilSMS or similar SMS gateway service
- **Purpose:** Send automatic text messages to vehicle owners
- **Cost:** Pay-per-message or monthly subscription
- **Estimated Cost:** ₱0.50 - ₱2.00 per SMS message

### 11. AI Service (Google Gemini API)
- **Service:** Google Gemini AI for image analysis
- **Purpose:** Detect vehicles and read license plates
- **Cost:** Pay-per-use (usually very affordable)
- **Estimated Cost:** ₱0.10 - ₱1.00 per image analyzed
